    Multi-Mode Arduino Bike Light

    Anemometer Circuit Design

    Made with: Circuit Design Sensor Calibration and Measurement Signal Processing Oscilloscope Data Analysis

    The goal of this project was to design a circuit to simulate an anemometer by measuring rotational speed of a 3D-printed propeller attached to a clamp pendulum sensor. It collected voltage readings at different rotation speeds to build a calibration curve linking voltage to rotational speed. Implementing a high-pass filtering and op-amp gain control to minimize noise and ensure precise measurements. Then Calculating the resistor and capacitor values to achieve desired cutoff frequency and amplification for accurate sensor output. From there analyzing the results using oscilloscope data and Bode plots to validate performance and establish linear relationships between speed and voltage.
